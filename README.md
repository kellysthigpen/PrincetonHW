## Contents

- `PS01.ipynb` - Main problem set notebook with carbon cycle models
- `PCA.ipynb` & `PCA_Facies.ipynb` - Principal Component Analysis
- `Covariances.ipynb` - Covariance analysis
- `2x2_3x3_SystemofEquations.ipynb` - Linear systems solving
- `2x2_vectors_plot.ipynb` - Vector visualization
- `Geology_transform_demo.ipynb` - Geological transformations
- `Plot_From_File.ipynb` - Data visualization examples
- `Solving_Linear_Systems.ipynb` - Linear algebra applications
